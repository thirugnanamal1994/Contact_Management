please follow the following steps to run the project 
1.install nodejs, expressjs, mysql
2. install wamp server(for database manipulations)
3. run wamp server
4. create a database named "contact_managment" in wamp server
5. invoke the file "contact_management.sql" from the root directory to the contact_managment db inthe wamp server
6. open nodejs command prompt
7.navigate to the extracted folder
8. npm install --g
9. npm start
10. open browser and "localhost:3000"
11. project is rendered in the browser
